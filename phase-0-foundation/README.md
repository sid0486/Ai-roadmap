# Phase 0 Foundation

A production-ready FastAPI backend built as part of my AI Engineer roadmap.

## Tech Stack
- Python 3.12
- FastAPI
- PostgreSQL
- Docker

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Run
```bash
uvicorn main:app --reload
```
